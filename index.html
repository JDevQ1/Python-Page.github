Introduction

Python is a high-level, interpreted programming language known for its readability and versatility.

It is widely used in web development, data science, automation, and more.

print("Hello, World!")

Python's syntax is clean and emphasizes code readability.

Python Basics

Variables in Python are dynamically typed, meaning you don't need to declare their type.

x = 10 # Integer name = "Alice" # String

Python supports basic operations like addition, subtraction, and more.

  • Addition: a + b
  • Subtraction: a - b
Data Structures

Python provides built-in data structures like lists, tuples, and dictionaries.

Lists are mutable and can hold mixed data types.

my_list = [1, "apple", 3.14]
  • Access elements with my_list[0]
Functions

Functions in Python are defined using the def keyword.

def greet(name): return f"Hello, {name}!"

You can call functions with arguments to perform tasks.

  • Example: greet("Alice")
Modules and Packages

Modules allow you to organize code into reusable files.

Packages are directories containing multiple modules.

import math print(math.sqrt(16)) # Outputs 4.0
  • Use import to access module functions.
